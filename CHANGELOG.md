# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Python 0.2.0] - 2026-03-09

### Added
- Keyed async queue and health monitor
- Full Python SDK sync with customer service POC
- ConversationMemory with pluggable backends
- ServiceBridge for remote function control
- RichReply with per-platform auto-degradation
- Streaming support across all adapters
- 18 channel adapters

### Changed
- Repositioned as lightweight embeddable messaging library

## [TypeScript 0.1.0] - 2026-03-09

### Added
- Full TypeScript SDK with ESM support
- 18 channel adapters matching Python parity
- Middleware pipeline (Commands, Access control, Rate limiting)
- ServiceBridge and ConversationMemory
- Vitest test suite

## [Java 0.1.0] - 2026-03-09

### Added
- Java 17+ SDK with Maven packaging
- 18 channel adapters matching cross-language parity
- Middleware pipeline with composable handlers
- JUnit 5 + Mockito test suite

## [MCP Server 0.1.0] - 2026-03-09

### Added
- Standalone MCP server supporting all 18 channels
- YAML configuration format
- Integration with AI agent frameworks via Model Context Protocol
